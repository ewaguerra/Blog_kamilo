Um design "moderno" não precisa de animações exageradas ou layouts complexos. Na prática, quatro pilares fazem a diferença.

## 1. Tipografia com hierarquia clara

Use uma fonte sans-serif para interface e mantenha contraste entre títulos e corpo. Títulos grandes com `letter-spacing` negativo transmitem confiança.

## 2. Espaço em branco generoso

Conteúdo apertado cansa. Margens amplas, cards com padding confortável e seções bem separadas melhoram a leitura imediatamente.

## 3. Paleta restrita

Escolha **um accent** (violeta, ciano, verde…) e use tons neutros para o resto. Gradientes sutis no hero ou no logo já criam identidade sem poluir.

> Menos cores = mais coerência visual.

## 4. Dark mode nativo

Em 2026, dark mode não é opcional. Implemente com CSS variables e respeite a preferência do sistema — depois deixe o usuário alternar manualmente.

## Checklist rápido

- [ ] Contraste WCAG AA no texto
- [ ] Layout responsivo mobile-first
- [ ] Hover states discretos
- [ ] Foco visível para acessibilidade

Design moderno é, antes de tudo, **clareza**.
